# Package for utility to make Go pointers

[![GoDev](https://img.shields.io/static/v1?label=godev&message=reference&color=00add8)][godev]

## Example

```go
import "github.com/go-ptr/go-ptr/ptr"

func main() {
	var str *string = ptr.NewString("hello")
	...
}
```
